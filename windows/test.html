MPV Protocol Handler Test

1. Standard Format

Basic format: mpvplay://[URL]
mpvplay://http://localhost:19798/static/http/localhost:19798/False/%2F115%2F123%2Ftest.mkv

2. Chrome 130+ Format

Chrome 130+ format (missing colon): mpvplay://http//[URL]
mpvplay://http//localhost:19798/static/http/localhost:19798/False/%2F115%2F123%2Ftest.mkv

3. Weblink Format

Weblink format: mpvplay://weblink?url=[URL]
mpvplay://weblink?url=http://localhost:19798/static/http/localhost:19798/False/%2F115%2F123%2Ftest.mkv

4. URL-Encoded Weblink Format

URL-encoded weblink format: mpvplay://weblink?url=[Encoded-URL]
mpvplay://weblink?url=http%3A%2F%2Flocalhost%3A19798%2Fstatic%2Fhttp%2Flocalhost%3A19798%2FFalse%2F%252F115%252F123%252Ftest.mkv
